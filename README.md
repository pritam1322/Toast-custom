### 🚀 Toast Notification Demo (Vite + React + TailwindCSS)
   This is a lightweight and customizable toast notification component built using Vite, React, Tailwind CSS, and Lucide React Icons.

### 📦 Tech Stack
        ⚡ Vite
        
        ⚛️ React
        
        🎨 Tailwind CSS
        
        🧩 Lucide Icons
        
        🆔 uuid for unique keys
